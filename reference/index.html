gaussxw(N)

Compute Gaussâ€“Legendre nodes and weights on [-1, 1].

Parameters:

Name Type Description Default
N int

Number of quadrature points.

 required

Returns:

Name Type Description
tuple

A tuple (x, w) where - x (ndarray): Nodes of the quadrature rule. - w (ndarray): Corresponding weights.

Examples:

>>> x, w = gaussxw(3)

gaussxwab(a, b, x, w)

Scale nodes and weights from the interval [-1, 1] to [a, b].

Parameters:

Name Type Description Default
a float

Left endpoint of the interval.

 required
b float

Right endpoint of the interval.

 required
x ndarray

Nodes on [-1, 1].

 required
w ndarray

Weights on [-1, 1].

 required

Returns:

Name Type Description
tuple

A tuple (x_scaled, w_scaled) where - x_scaled (ndarray): Scaled nodes in [a, b]. - w_scaled (ndarray): Scaled weights for [a, b].

Examples:

>>> x, w = gaussxw(2)
>>> x_s, w_s = gaussxwab(1, 3, x, w)
>>> len(x_s), len(w_s)
(2, 2)
